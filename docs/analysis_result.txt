The returned result for marking the essay
score - the marked score of the essay
category - the classification of the essay such as a descriptive essay etc.
raw - contains the extracted values from the essay placed in an object
    -> analysis - the value the semantic analysis of the essay
    -> characters - the total number of characters in the essay
    -> misspelt_words - the total number of misspelt_words
    -> punctuation - total number of punctuation marks in the essay
    -> sentences - total number of sentences in the essay
    -> vocabulary - contains an object of the vocabulary in the essay
        -> CC - coordinating conjunction
        -> CD - cardinal digit
        -> DT - determiner
        -> EX - existential there (like: "there is" ... think of it like "there exisits")
        -> FW - foreign word
        -> IN - preposition/subordinating conjunction
        -> JJ - adjective 'big
        -> JJR - adjective, comparative 'bigger'
        -> JJS - adjective, superlative 'biggest'
        -> LS - list marker 1)
        -> MD - modal could, will
        -> NN - noun, singular 'desk'
        -> NNP - proper noun, singular 'Harrison'
        -> NNPS - proper noun, plural 'Americans'
        -> NNS - noun, plural 'desks'
        -> PDT - predeterminer e.g 'all the kids'
        -> POS - possessive ending e.g parent's
        -> PRP -  personal pronoun e.g I, he, she
        -> PRP$ - possessive pronoun e.g my, his, hers
        -> RB - adverb e.g very, silently
        -> RBR - adverb, comparative e.g better
        -> RBS - adverb, superlative e.g best
        -> RP - particle give up
        -> TO - to go 'to' the store
        -> UH - interjection errrrrrm
        -> VB - verb, base from take
        -> VBD - verb, past tense took
        -> VBG - verb, gerund/present participle taking
        -> VBN - verb, past participle taken
        -> VBP - verb, sing. present, non-3d take 
        -> VBZ - verb, 3rd person sign. present takes
        -> WDT - wh-determiner which
        -> WP - wh-pronoun who, what
        -> WP$ - possessive wh-pronoun whose
        -> WRB - wh-adverb where, when
