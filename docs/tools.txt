Tools used
- Python(scikit-learn) - for building and training the model
- Javascript(React) - for building the UI interface for the application 
- Flask - for building the server for user interaction between the application and the model
	

Libraries used
- sklearn - has various modules and algorithms which has been optimized
- nltk - was used for text analysis and useful for converting text into numbers
- pyenchant - has error checking functions useful for analyzing the language control of the essay
